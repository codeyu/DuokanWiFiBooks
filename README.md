# DuokanWiFiBooks
基于 **node.js** 实现 **多看WiFi传书** 功能

可作为 **文件管理系统** 使用

使用 [diet.js](http://dietjs.com/) 框架实现

## 本地运行

git clone https://github.com/codeyu/DuokanWiFiBooks.git

cd DuokanWiFiBooks

npm install

npm start

浏览器打开：[http://localhost:3000/](http://localhost:3000/)
